## Overview
&nbsp;&nbsp;&nbsp;The **Coffee Order Manager project** is a Python web application built with **Flask** to provide an intuitive interface for managing coffee orders. The application employs a **layered architecture** and integrates an **SQLite database** to efficiently store and retrieve order details. Users can create, edit, and manage orders, including itemized breakdowns of products, quantities, and prices.

## Features

### 1. **Order Management**
   &nbsp;&nbsp;&nbsp;‚Ä¢ The application implements essential **CRUD operations** to manage orders. Users can create, view, update, and delete orders, with seamless integration between the database and user interface to ensure efficient data management and interaction.

### 2. **Validation and Error Handling**
   &nbsp;&nbsp;&nbsp;‚Ä¢ **Input Validation**: Ensures that all inputs, such as item quantities, are valid and within acceptable ranges.  
   &nbsp;&nbsp;&nbsp;‚Ä¢ **Error Messages**: Friendly error messages guide users when invalid data is entered or unexpected issues occur.

### 3. **Responsive User Interface**
   &nbsp;&nbsp;&nbsp;‚Ä¢ Clean, minimalistic HTML templates make navigation straightforward and user-friendly.  
   &nbsp;&nbsp;&nbsp;‚Ä¢ The coffee menu and order summaries are dynamically rendered using **Jinja2** templates.

### 4. **Database Integration**
   &nbsp;&nbsp;&nbsp;‚Ä¢ **SQLite Database**: Stores all orders and their associated items, enabling persistence and retrieval across sessions.

## Technologies Used:
   &nbsp;&nbsp;&nbsp;‚Ä¢ **Python** and **Flask** for application logic and routing.  
   &nbsp;&nbsp;&nbsp;‚Ä¢ **SQLite** for lightweight and persistent data storage.  
   &nbsp;&nbsp;&nbsp;‚Ä¢ **Jinja2** for dynamic HTML template rendering.
